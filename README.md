
# Movement and camera trap simulations

This simulation study is part of work by [Goldsborough et al., (in
prep)](https://doi.org/10.1101/2025.04.08.647785).
In this simulation, we make various particles move following Vicsek dynamics,
adjusted such that the simulated groups can each move (a) as one entity,
(b) as two non-interacting subgroups, or (c) all individuals separately.
